An analogy to describe JavaScript and its relationship to HTML and CSS.

Hyper Text Markup Language (HTML) is the basic building blocks of a webpage It is a text-based approach to describing how content (elements like headings, text, lists) should be displayed on a page.

Cascading Style Sheets (CSS) is a language that describes how HTML element should be displayed in terms of styling. Styling can include color, font, position of elements, spacing between elements, etc.

Javascript (JS) is a language that allows content to be dynamic such as animations and any movement of elements or refreshes on the webpage.

HTML and CSS are like a couple in a relationship. HTML wears the pants, CSS is the good looking one. Then they meet JS, a super-active, into extreme sports kind of person. They enter into a polyamourous relationship with JS and together they go dancing. 😅

What is the DOM? and an example of how you might interact with it.

The Document Object Model (DOM) is a JS representation of a webpage. It's our JS "window" or access portal into the contents of a webpage. It's just a bunch of objects that you can interact with via Javascript.

The document object is the entry point into the world of the DOM. It contains representations of all the content on a page, plus a lot of useful methods and properties. The Document contains objects that represent the content on the page.

Within the document, are various methods and properties which can be used to manipulate the document. There are methods that allow us to select elements, using JS to single out an element, or elements with a certain class
